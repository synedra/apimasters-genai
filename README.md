# RAG and Prompt Engineering: From Concepts to Creation
### An APImasters Course by Kirsten Hunter

This repository contains the notebooks and worksheets for the APImasters course **RAG and Prompt Engineering: From Concepts to Creation** — a practical guide to working with large language models, from understanding how they work to building your own RAG pipeline.

---

## Module 2: Prompt Engineering Essentials

### Worksheet: Common Prompting Mistakes and How to Fix Them

Work through six hands-on prompt challenges covering the most common prompting mistakes — and how to fix them fast.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/synedra/apimasters-genai/blob/main/notebooks/prompt_engineering_worksheet.ipynb)

The notebook uses **Groq** by default — free, no credit card required. Sign up at [console.groq.com](https://console.groq.com) to get your API key before opening the notebook.

**Challenges covered:**
- Too vague — clowns
- Too much at once — goth teenager
- No format specified — recipe
- Forgetting the audience — GLP-1
- Not iterating — pitching
- Bonus: put it all together — climate change
